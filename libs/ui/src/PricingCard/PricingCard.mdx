# PricingCard

The `PricingCard` component is a reusable UI element.

## Usage

### Basic Example

<Canvas of={PricingCardStories.Free} />

```tsx

  Content
</PricingCard>
```

## Accessibility

The `PricingCard` component follows WAI-ARIA guidelines:

- Supports keyboard navigation
- Includes proper ARIA attributes
- Compatible with screen readers

### Keyboard Interactions

| Key | Description |
|-----|-------------|
| `Tab` | Moves focus to the component |
| `Enter` | Activates the component |
| `Space` | Activates the component |
