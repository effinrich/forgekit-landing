# Features

The `Features` component is a reusable UI element.

## Usage

### Basic Example

<Canvas of={FeaturesStories.Default} />

```tsx
<Features>
  Content
</Features>
```

## Accessibility

The `Features` component follows WAI-ARIA guidelines:

- Supports keyboard navigation
- Includes proper ARIA attributes
- Compatible with screen readers

### Keyboard Interactions

| Key | Description |
|-----|-------------|
| `Tab` | Moves focus to the component |
| `Enter` | Activates the component |
| `Space` | Activates the component |

## Composition

### With Other Components


## Responsive Design

Use responsive props for different screen sizes:

```tsx

  Responsive Features
</Features>
```
